Practical 3D Data Preservation

Sustain 3D is a clear step-by-step guide for teams that create 3D data and need durable organization, metadata, storage planning, and policy decisions without a large dedicated preservation staff.

Start the Workflow -> Go to Repository Setup

Who this is for

  • University labs
    Teams generating scans that need a repeatable project structure.
  • Small museums
    Institutions digitizing collections with limited technical staffing.
  • Libraries and archives
    Programs adapting 2D preservation workflows to 3D packages.
  • Community heritage projects
    Groups preserving local or cultural materials with practical guardrails.

What this site does

1) Gives one clear flow

Home -> Workflow -> Repository Setup -> Files -> Metadata -> Storage -> Policies -> Resources.

2) Defines what to do at each step

Each page starts with a goal and where the step fits in the full project lifecycle.

3) Separates master and access outputs

Preservation masters are handled as long-term assets while access copies are delivery files.

4) Makes policy and ethics visible early

FAIR and CARE appear at workflow start so technical and ethical decisions happen before publishing.

The core idea in three steps

  1. Structure your projects. Use the same repository and project pattern every time.
  2. Describe your work. Capture metadata and paradata as you move through each stage.
  3. Control distribution. Keep masters safe, publish access copies, and document rights.
