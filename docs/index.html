Practical 3D Data Preservation

Sustain 3D is a public, step-by-step web resource for teams that create 3D data and need durable organization, metadata, storage planning, and policy decisions without a large dedicated preservation staff.

Start the Workflow -> Go to Repository Setup

Who this is for

  • University labs
    Teams generating scans that need a repeatable project structure.
  • Small museums
    Institutions digitizing collections with limited technical staffing.
  • Libraries and archives
    Programs adapting 2D preservation workflows to 3D packages.
  • Community heritage projects
    Groups preserving local or cultural materials with practical guardrails.

What this site does

1) Gives one clear path

Home -> Workflow -> Repository Setup -> Files -> Metadata -> Storage -> Policies -> Resources -> In Practice.

2) Defines actions at each step

Each page provides practical workflow steps, examples, and decision support for implementation.

3) Scales from scans to print files

The same structure works for photogrammetry/LiDAR outputs, born-digital models, and 3D print deliverables.

4) Keeps preservation and access separate

Preservation masters stay controlled for long-term stewardship while derivatives are prepared for sharing and use.

The core idea in three steps

  1. Structure your projects. Use the same repository and project pattern every time.
  2. Describe your work. Capture metadata and paradata as you move through each stage.
  3. Control distribution. Keep masters safe, publish access copies, and document rights.
